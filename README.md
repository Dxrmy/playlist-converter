# Playlist Converter

Here is a quick script to download and run the playlist converter.

> **Note:** I have personally only tested this for transferring from **Spotify to YouTube Music**. I have not tried it the other way around yet!

**Windows:**
```powershell
irm https://raw.githubusercontent.com/Dxrmy/playlist-converter/main/run.ps1 | iex
```

**macOS & Linux:**
```bash
curl -sL https://raw.githubusercontent.com/Dxrmy/playlist-converter/main/run.sh | bash
```
